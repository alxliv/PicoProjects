# Several Raspberry Pi Pico 2(w) projects for study and fun

## Current list of projects

1. [async_tasks](./async_tasks/README.md)
   
	A tiny cooperative task scheduler with Pico example that runs non-blocking callbacks at set intervals.
    
2. **blink_simple**
   
    Minimal Raspberry Pi Pico demo — alternately blinks two LEDs and prints a counter to stdio.
